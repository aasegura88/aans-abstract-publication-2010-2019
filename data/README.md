# Data (not included)

This repository does not include the analysis-ready dataset.

To reproduce the analyses, place the analysis-ready CSV at:

- data/AANScleanedv3 JMR 2.25.csv

or set an environment variable:

- AANS_DATA_PATH="/path/to/AANScleanedv3 JMR 2.25.csv"

The dataset was derived from publicly available AANS meeting abstract archives and PubMed-indexed publications.
